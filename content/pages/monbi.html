MONBI Tracing System

Background

The MONBI Tracing System project has emerged from the pressing need within the company to address the growing complexity of expense management. Prior to the existence of this web-based system, the company relied on manual methods involving manual data input and storing information in Excel spreadsheets. This manual process, aside from being time-consuming, posed the risk of human errors that are often difficult to detect. Inefficient management of expense records has created significant obstacles and challenges in the company's operations. To address these hindrances, the company recognized the need for an efficient, automated, and reliable system to simplify and enhance expense management.

The MONBI Tracing System project represents a critical step in the company's efforts to improve their expense management processes. In an increasingly competitive business environment, having a strong control over expense records is crucial for optimizing efficiency and profitability. Therefore, this project serves as a proactive response to changing market dynamics and the demand for better expense management.

The project also aligns with the rapid advancements in information technology. Implementing a web-based technology framework using Laravel with PHP programming language and MySQL as the database management system is a wise step in confronting the increasing complexities of the modern business world. In this context, the MONBI Tracing System project plays a crucial role in helping the company embrace the digital era with preparedness.

Poroject Objectives

The primary objective of this project is to create a web-based solution that automates and streamlines the expense tracking and management process for the company. In this regard, the project aims to:

  1. Enable users to easily upload their expense data.
  2. Automate the calculation and processing of data accurately.
  3. Build an interactive dashboard for real-time expense monitoring.

By achieving these objectives, the project is expected to deliver significant benefits in optimizing the company's expense processes.

Technologies Used

  • Laravel 8.0
  • PHP Programming Language
  • Bootstrap 5.0
  • MySQL
  • XAMPP
  • Visual Studio Code
